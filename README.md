# Northstar IT Website

A local IT business website with a public services page, booking form, and business dashboard.

## Open Locally

Open `index.html` in a browser.

## Included

- `index.html` - Website structure
- `styles.css` - Visual design and responsive layout
- `script.js` - Booking form and dashboard behavior
- `assets/it-hero.png` - Website hero image

## Notes

Bookings are stored in the browser using local storage. This keeps the demo simple and portable without requiring a server.
